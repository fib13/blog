HTML Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

Syntax

<tagname style="property:value;">

Backgroud Color

十三时记

无他，惟手熟尔


<div style="background-color:powderblue;">
    <h1>十三时记</h1>
    <p>无他，惟手熟尔</p>
</div>
注意
因防止全局渲染，因此将 body 元素换成 div 元素。
